# 🚀 React Demo App (Counter + Form)

## 📌 Project Description
This is a simple React application built using **Vite and Tailwind CSS**.  
The project demonstrates two core features:
- A **Counter App** using React state
- A **Form** with input handling using React events

The main goal of this project is to understand **state management, event handling, and component structure in React**.

---

## 🛠️ Technologies Used
- React.js
- Vite
- Tailwind CSS
- JavaScript (ES6+)

---

## 📂 Features
### 🔢 Counter
- Increase counter value
- Decrease counter value
- Reset counter to zero

### 📄 Form
- Input fields for Name and Email
- Controlled components using `useState`
- Form submission using `onSubmit`
